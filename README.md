# STM32F103C8T6 Custom Development Board

A custom STM32F103C8T6 development board designed using KiCad 9 for embedded system development and hardware prototyping.

## Features

- STM32F103C8T6 (ARM Cortex-M3)
- USB Micro Power Interface
- AMS1117-3.3V Regulator
- SWD Programming Header
- UART Interface
- I²C Header
- 16 MHz Crystal Oscillator
- BOOT0 Configuration
- Compact 2-layer PCB


## Tools Used

- KiCad 9
- STM32CubeMX
- STM32CubeIDE
