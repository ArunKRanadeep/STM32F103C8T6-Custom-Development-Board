# STM32F103C8T6 Custom Development Board

Designed My Own Custom STM32 Development Board! 

Instead of using a commercial STM32 development board, I designed and developed a compact STM32F103C8T6 custom development board using KiCad 9.

The objective was to create a reliable and compact hardware platform for embedded system development by integrating the essential peripherals required for programming, debugging, and communication.

  Key Features
STM32F103C8T6 (ARM Cortex-M3)
Compact 2-layer PCB
USB Micro connector for power and USB communication
AMS1117-3.3V LDO regulator
SWD programming/debug interface
UART communication header
I²C expansion header
16 MHz crystal oscillator
BOOT0 selection switch
VDDA analog power filtering
  Design Validation

* ERC Passed

* DRC Passed

* 3D PCB verified

Tools Used :-
KiCad 9
STM32CubeMX
STM32CubeIDE

This project enhanced my skills in embedded hardware design, schematic capture, PCB layout, power supply design, and STM32-based embedded systems.
